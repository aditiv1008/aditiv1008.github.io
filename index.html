Cattelite Terrain Odyssey

AI Book Recommenders

Python, TensorFlow, Flask, Streamlit, HTML, CSS, JavaScript

I implemented two collaborative filtering book recommendation systems, one using item-based and one using user-based filtering.
First Approach:
• Created a graphical interface using Streamlit for users to pick a book and generate recommendations as shown in the video.
• Implemented a nearest neighbors heuristic based on a user-item matrix, finding the nearest neighbors of a book in the user-item space for generating recommendations..
Second Approach:
• Explored TensorFlow, an open-source machine learning framework that works with Keras as its backend, facilitating the development and training of a neural network-based recommendation system.
• Utilized embedding layers for users and items in neural networks to learn non-linear relationships and capture intricate patterns in user-item interactions.
• Employed a neural network with multiple layers to learn non-linear mappings between users and items, addressing the struggles of distance-based Nearest Neighbors.
• Trained neural networks using backpropagation, adjusting embeddings and biases to minimize the difference between predicted and actual ratings.
• Applied the sigmoid activation function to introduce non-linearity and capture the non-linear nature of user preferences.
• Implemented regularization techniques, including dropout layers, to prevent overfitting and address the issue of repeated recommendations.
• Implemented UI using Flask, HTML, CSS
